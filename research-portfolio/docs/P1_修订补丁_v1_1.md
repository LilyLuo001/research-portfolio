## 修订 2|新增 T2a:前置模拟功效分析(kill-switch 预演)

**位置**:插在 T1 与 T2 之间;**指派**:Claude Code(Sonnet,一个 prime block 内完成);**动因**:v1.0 的 kill-switch 第 2 关(ConvExp 分布)要到第 6 周 WRDS 管道建成后才触发;用 FEDS Note 公开数字 + 粗粒度先验可以在第 1 周以百分之一的成本预演同一判定,若明显不过关,整条 WRDS 管道的预算与时间全部省下。

```
[粘贴 CONTEXT PACK]
任务(T2a): 在不接入 WRDS 的前提下做模拟功效分析。
输入: events_merged.csv(若 T1 未完成, 用 Saglam-Tuzun Note 附表的事件清单代替,
逐条附页码); Note 中公开的转换基金 AUM 与持仓集中度描述性统计(逐条附页码)。
步骤:
1. 用公开 AUM × 典型持仓权重先验, 模拟 ConvExp_i,e 的合理分布区间
   (乐观/基准/悲观三档), 明确标注每个先验假设。
2. 在三档分布下, 模拟 stacked DiD 的 MDE: 给定文献中盈余信息纳入变量
   (FERC/PEAD)的典型效应量与截面方差(只允许引用文献包内数字, 附出处),
   计算各档下检出 0.5σ/1σ 效应所需的处理组规模。
3. 输出 power_memo.md: 三档结论 + "若真实 ConvExp 分布落在悲观档,
   建议触发 kill-switch"的明确判定线。
规则: 所有模拟代码落盘、种子固定; 每个输入数字附页码或文献包定位符;
先验无依据处输出 NEED_HUMAN。此备忘录只用于早期止损判断, 不进论文。
```

**联动**:§6 节奏表中 T2a 排第 1 周;T2(WRDS 管道)改为"T2a 结论非悲观档"后启动。

## 修订 3|T5 规格新增 Russell 重构混淆处理(强制项)

DFA 锚波次生效日(2021-06-11)紧邻 2021 年 6 月 Russell 指数年度重构,后者在 DFA 重仓的小盘/价值股上机械性地改变成交量、价差与指数基金持仓——与本文结果变量直接重叠。v1.0 的红队目标清单(T10)列出了 DFA 单家族与 2021-06 并发事件,但未点名 Russell 重构这一最具体、referee 一眼能看到的通道。

**T5 规格 prompt 追加以下条目**(插入"三套推断"之前):

```
- Russell 重构混淆: 对锚波次样本, 构造股票层面的 reconstitution 暴露变量
  (当年是否被加入/剔除/在 Russell 1000/2000 间迁移, 数据口径由规格给出),
  主规格加入该控制; 另给一个"剔除重构受影响股票"的子样本规格与一个
  ConvExp × 重构暴露交互规格, 三者并列报告。规格书须论证: 若主效应在
  剔除样本中消失, 结论如何降级表述。
```

**T10 红队 prompt 的"特别攻击"清单追加**:`2021-06 Russell 重构与 DFA 波次的机械重叠`。

**robustness_grid.csv(T7)追加**对应的 2–3 行变体配置。

## 修订 4|新增 T13(P1):ANT 对照样本——披露机制与交易机制的分离检验

**动因**:"ETF 化"同时改变两件事:日内可交易的外壳 + 每日持仓披露。半透明主动 ETF(ANT/semi-transparent,如 Fidelity/Invesco 的 NYSE AMS、Precidian ActiveShares 结构)改变外壳但不做每日全透明披露,构成天然的机制分离样本。哪怕只做一个小样本对比,论文就从"无法区分机制"升级为"带机制检验",这是中游与上游期刊的分水岭。

**指派**:事件采集沿用 T1 双通道模式(Kimi + Gemini Flash 免费档);分析并入 T5/T6 框架,不新建管道。

```
[粘贴 CONTEXT PACK]
任务(T13): 构建 ANT(半透明主动 ETF)对照事件集。
范围: 2019-11(SEC 批准 ActiveShares)以来上市或由共同基金转换而来的
半透明主动 ETF; 与 T1 相同的 schema, 增加两列:
  disclosure_regime(daily_full / semi_transparent)| proxy_basket_type
数据来源: 我提供的 EDGAR 文件包(N-1A/497 中的披露结构描述)。
规则同 T1: 逐字段定位符、双通道、NA 不推断。
下游: T5 增加一个"转换为全透明 vs 转换/上市为半透明"的三重差分规格
(由 T5 规格通道在裁决时一并设计); 若 ANT 样本量 < 15 只, 降级为
描述性对比小节, 不做推断, 并在论文 limitation 中说明。
```
