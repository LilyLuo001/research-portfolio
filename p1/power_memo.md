## 1. 设计与先验

处理剂量 dose_i = Σ_f AUM_f · w_if / mktcap_i (bps of market cap)。
估计器 = shared/econlib `stacked_did` (P1 主规格), 窗口 ±4 季度, 单一锚波次。
先验一览 (全部粗粒度, 全部可被 WRDS 实测替换):

| 先验 | 内容 | 依据 |
|---|---|---|
| PRIOR-1 | 可投资截面 ~3,000 名, 市值 log-normal(ln 2e9, 1.6) | 尺度锚, 结论以相对量表述 |
| PRIOR-2 | 基金风格宇宙按市值秩切片 (large/all/small/mid-value) | 基金名称与公开风格描述 |
| PRIOR-3 | 宇宙内权重 ∝ cap^γ × lognormal(0,0.5) 抖动; γ = 情景旋钮 (0.70/0.85/1.00) | DFA 为倾斜准主动组合; γ<1 = 小盘倾斜 |
| PRIOR-4 | 处理判定阈值 θ = 25bps (敏感性 10/50bps) | 无依据的自由参数 — **NEED_HUMAN: θ 的合理区间需 owner 确认** |
| PRIOR-5 | 噪声 iid N(0,1); 报告的 MDE 已乘 1.5 保守系数 (未估计, 标注) | iid 是乐观噪声模型; 序列相关/共同冲击会恶化功效 |

## 2. 三档结论

保守 MDE80 (含 ×1.5 系数; N_T = dose≥θ 的处理股数, N_C = 其余):

| 档 (γ) | θ=10bps | θ=25bps | θ=50bps |
|---|---|---|---|
| 乐观 (0.70) | N_T 2608 / **0.153σ** | N_T 2056 / **0.111σ** | N_T 1039 / **0.108σ** |
| 基准 (0.85) | N_T 2563 / **0.146σ** | N_T 1936 / **0.108σ** | N_T 639 / **0.126σ** |
| 悲观 (1.00) | N_T 2459 / **0.134σ** | N_T 1779 / **0.105σ** | N_T 357 / **0.159σ** |

**三档全部通过**: 最差单元格 MDE80 = 0.159σ, 相对 prompt 的 0.5σ / 1σ 检出
目标有 3–6 倍余量。估计器校验: econlib stacked_did 子抽样仿真功效
(δ=0.2σ → 0.787) 与解析曲线 (0.82) 一致。

**结构性发现 (对所有档稳健):**
1. **锚波次的处理规模不是瓶颈。** $28.65bn 摊到任何合理的风格宇宙上,
   在 θ=25bps 下处理组都有数百到两千只股票, 保守 MDE80 ≈ 0.1σ 量级 —
   远优于 prompt 设定的 0.5σ / 1σ 检出目标。
2. **真正的约束在对照组一侧。** DFAC 一只基金就覆盖 ~2500/3000 名;
   θ 取得越低, "干净对照"越稀缺 (θ=10bps 时对照仅 ~300 名, MDE 反而变差)。
   这与 AMEND P1-3 的 Russell 重构混淆叠加 — 对照池还要再剔除重构受影响
   股票。**T5 规格必须显式报告干净对照的计数。**
3. **悲观档的真实形态**不是"检不出 0.5σ", 而是"纯市值加权 + 高剂量要求"
   使高剂量处理组塌缩 (γ: 0.70→1.00 时 θ=50bps 单元的 N_T 从 1039 缩至
   357; 校正 2026-07-10, 审计 K-2 — 原稿误写 "N_T→0", 那是修正前无抖动
   模型的退化行为)。若真实持仓接近纯市值加权, 剂量分布退化为常数,
   剂量异质性识别 (连续处理强度规格) 失效, 只剩二元处理对比。

## 3. kill-switch 判定线

模拟给出的最小可行规模 (保守 MDE80 ≤ 目标; 处理/对照中较小一侧):

- 检出 **0.5σ**: 处理/对照中较小一侧 ≥ **33** 只
- 检出 **1.0σ**: 较小一侧 ≥ **8** 只

**判定线 (T2 WRDS 管道建成、真实 ConvExp 分布落地后立即核对):**

> 若真实 ConvExp 分布满足以下任一条, 判定落入悲观档, 建议触发 kill-switch,
> 取消 WRDS 后续预算线:
> 1. dose ≥ 25bps 的股票数 < 33 (0.5σ 不可检出);
> 2. 干净对照 (dose < 10bps 且未被 Russell 重构影响) 的股票数 < 33;
> 3. dose 截面分布近似退化 (P90/P50 < 1.5) 且中位数 < 25bps —
>    剂量异质性识别失效, 且二元对比的处理定义变得任意。
>
> 若仅条件 3 触发而 1、2 不触发: 降级为"只报告二元处理主规格", 不 kill。

## 4. CITE_REQUEST / NEED_HUMAN

- **CITE_REQUEST**: FERC、PEAD 的典型效应量与截面方差 (文献包条目) — 用于
  把上表 σ 单位映射到文献数量级, 判断 0.1σ 的 MDE 在经济意义上是否苛刻。
- **NEED_HUMAN**: PRIOR-4 阈值 θ 的合理区间; FEDS Note 附表的逐基金持仓
  集中度数字 (owner 手动下载 Note 后可直接替换 PRIOR-2/3, 重跑脚本即可,
  种子固定, 结论可比)。
- 校验项: econlib `stacked_did` 在子抽样面板上的仿真功效与解析曲线一致
  (results JSON `verification_subsample`), 排除估计器实现问题。
